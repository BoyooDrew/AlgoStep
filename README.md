# AlgoStep

> 面向 CSP 考生的算法可视化教学工具
> Algorithm Visualization Tool for CSP Exam Preparation

## 简介

AlgoStep 是一个基于 Web 的算法可视化平台，专为信息学奥赛（CSP-J/S）考生和 C++ 入门学习者设计。通过交互式步骤演示，帮助学生直观理解经典算法的执行过程。

## 技术栈

- 纯 HTML5 / CSS3 / JavaScript（无框架依赖）
- 深色主题设计系统
- 响应式布局

## 在线访问

[https://algo-step.surge.sh](https://algo-step.surge.sh)

## 本地使用

```bash
# 克隆仓库
git clone <your-repo-url>
cd algostep

# 用浏览器打开首页
open index.html
```

## 部署

本项目为纯静态网站，支持任何静态托管服务（GitHub Pages、Gitee Pages、Surge.sh 等）。
